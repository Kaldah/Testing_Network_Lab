utils.lab_manager module¶

Lab Manager for StormShadow.

This module provides centralized lab environment management by managing Docker containers and lab infrastructure.

class LabManager[source]¶

Bases: object

Manages lab module lifecycle and coordination.

This manager provides: - Lab module discovery and loading - Lab lifecycle management (start/stop/status) - Coordination with other system components
