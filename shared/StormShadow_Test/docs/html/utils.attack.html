utils.attack.attack_manager module¶

Attack Manager for StormShadow.

This module provides centralized attack lifecycle management by directly working with attack modules and a simple discovery mechanism.

class AttackManager[source]¶

Bases: object

Manages attack module lifecycle and coordination.

This manager provides: - Attack module discovery and loading - Attack lifecycle management (start/stop/status) - Coordination with other system components
